About MySmartHome

This web application was developed by Jonathan. It is a dynamic web application that employs routes, forms, and database access to present a dashboard that enables a customer to control devices in a user-friendly interface. Users are able to add, delete, update, and view the status of the devices that have been added to the system. The user interface provides a pre-defined list of devices and appliances that are available for selection and customizable by name.
